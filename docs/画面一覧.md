# PJ_NOTEBOOK 画面一覧

最終更新: 2026-07-20
対象システム: `outputs/system-design-basics-ja.html`（学習ジャーナル + 設計Wiki）

> 本システムは単一HTMLの **SPA（シングルページアプリ）** です。個別ページ遷移ではなく、
> 1つの画面内で「表示モード（`state.mode`）」「ワークスペース種別」「オーバーレイ」を
> 切り替えて動作します。本書では利用者視点の“画面”を、これらの状態単位で一覧化します。

---

## 0. 画面構成の全体像

```
index.html（入口・リダイレクト）
└─ system-design-basics-ja.html（本体SPA）
   ├─ 共通レイアウト
   │   ├─ ヘッダー / 検索バー
   │   ├─ 左: 本棚・Bookツリー（モバイルはドロワー）
   │   ├─ 中央: 本文 / フォーム
   │   └─ 右: 関連record・バックリンク・タグ
   ├─ 表示モード（S-10〜S-16）
   ├─ 特殊ワークスペース画面（S-20〜S-24）
   ├─ 学習コンテンツ画面（S-30）
   └─ オーバーレイ: メニュー / プレビュー / トースト（S-40〜S-52）
```

---

## 1. 入口・共通レイアウト

| ID | 画面名 | 種別 | 概要 |
|---|---|---|---|
| S-00 | 入口ページ | リダイレクト | `index.html`。本体HTMLへ自動遷移する入口 |
| S-01 | ライブラリ / 本棚ホーム | レイアウト領域 | 本棚切替タブ・カテゴリ・Book一覧を表示する最上位のシェル（`.library-shell`） |
| S-02 | Wikiワークスペース | レイアウト領域 | 左（本棚/Bookツリー）・中央（本文）・右（関連情報）の3ペイン基本レイアウト（`.wiki-workspace`） |
| S-03 | 検索 | 共通部品 | 検索入力とサジェスト表示。Book・章・本文・タグへ直接移動（`searchResults`） |

---

## 2. 表示モード画面（`state.mode` による切替）

中央ペインの状態。閲覧を基本とし、明示的に編集・作成モードへ切り替える。

| ID | 画面名 | `state.mode` | 概要 | 主な操作 |
|---|---|---|---|---|
| S-10 | Page閲覧 | `view` | 章/Pageの本文を読む標準状態 | 読む・アノテーション参照・表ソート・関連へ移動 |
| S-11 | Page編集 | `edit-page` | ブロックエディタで本文を編集 | ブロック挿入/変換/複製/削除/並び替え・スラッシュコマンド・注釈 |
| S-12 | 新規章作成 | `new-page` | 所属Bookを選び章タイトル・本文を入力 | 作成・キャンセル |
| S-13 | 新規Book作成 | `new-book` | Book名・短縮名・アイコン・色・説明を登録 | 作成・キャンセル |
| S-14 | Book編集 | `edit-book` | 上記＋所属本棚の修正 | 保存・キャンセル |
| S-15 | 新規本棚作成 | `new-shelf` | 本棚名・アイコン・説明を登録 | 作成・キャンセル |
| S-16 | 本棚編集 | `edit-shelf` | 本棚名・アイコン・説明を修正 | 保存・キャンセル |

## 3. 特殊ワークスペース画面（共有record系）

共通の `records` を各観点で表示する専用ページ。

| ID | 画面名 | 概要 | 主な操作 |
|---|---|---|---|
| S-20 | 共有Database | 全recordを一覧・管理するテーブルビュー（`renderSharedDatabaseView`） | ソート・フィルタ・グループ化・type編集 |
| S-21 | Tasks | `task` typeを持つrecordを状態別に管理。**Todo / Doing / Done / Hold**（`renderTasksView`） | 状態変更・タスク操作 |
| S-22 | Dashboard | 共有Databaseを集計し未整理・疑問・タスク・用語を俯瞰（`renderDashboardView`） | 状況確認・該当recordへ移動 |
| S-23 | 学習ジャーナル / 今日の記録 | 当日分のジャーナルページ（自動生成）。日々の気づき・メモ・疑問を記録（`ensureTodayJournalPage`） | 記録追加・あとで整理 |
| S-24 | 未整理Inbox | 分類前recordの受け皿（`ensureInboxPage`） | メモ確認・Page/Book化・タスク化・移動 |

## 5. オーバーレイ（メニュー・プレビュー・通知）

対象の近くに表示するポップアップ類。hover / 右クリック / `...` の内容を一致させる。

| ID | 画面名 | 種別 | 概要 |
|---|---|---|---|
| S-40 | 作成メニュー | ポップアップ | 本棚 / Book / 章 / メモ等の新規作成導線（`showCreateMenu`） |
| S-41 | 設定メニュー | ポップアップ | JSON入出力・初期化などの設定操作（`showSettingsMenu`・`exportJson`・`resetToInitialState`） |
| S-42 | 本棚メニュー | ポップアップ | 本棚の切替・追加・改名・削除（`showShelfMenu`） |
| S-43 | Book管理メニュー | コンテキストメニュー | Bookの改名・複製・移動・削除（`showBookManagementMenu`） |
| S-44 | Page管理メニュー | コンテキストメニュー | 章/Pageの改名・複製・移動・削除（`showPageManagementMenu`） |
| S-45 | ノートメニュー | コンテキストメニュー | メモのPage化・移動・ToDo化・DB化・type編集（`showNoteMenu`） |
| S-46 | ブロックメニュー | コンテキストメニュー | ブロックの複製・削除・移動・種別変換（`showBlockMenu`） |
| S-47 | アノテーションメニュー | ポップアップ | 注釈の色変更・メモ編集・削除（`showAnnotationMenu`） |
| S-50 | テンプレートプレビュー | プレビュー | ドキュメントテンプレートの詳細プレビュー（`openPreview` / `renderTemplatePreview`） |
| S-51 | ブロックプレビュー | プレビュー | 挿入前のブロック外観プレビュー（`showBlockPreview`） |
| S-52 | トースト通知 | 通知 | 保存・変換などの操作結果を一時表示（`showToast`） |

## 7. 補足

- 画面遷移は URL 遷移ではなく `state.mode` とアクティブ本棚/Book/Pageの変更で行われ、
  変更のたびに `render()` が再描画する。
- 各状態は `localStorage`（`systemDesignWiki.v1`）に保存され、再読込後も復元される。
- 印刷・専用エラー画面などの独立画面は現状なし（SPA内の状態で完結）。
